# МАТЕМАТИЧЕСКАЯ ОСНОВА PRIMEWALKER

## Алгоритм построения

### Входные данные:
- `N` - количество простых чисел для анализа (начиная с 7)
- `scale` - масштаб отображения
- `imageSize` - размер выходного изображения

### Шаги построения:

1. **Генерация последовательности**
```
P = [7, 11, 13, 17, 19, 23, 29, 31, 37, ...]
```


2. **Фильтрация старших близнецов**
- Для каждого простого числа p проверяем: является ли (p-2) простым и присутствует ли в P?
- Если да, то p (старший близнец) исключается
- Остаются: 7, 11, 17, 23, 29, 37, ...

3. **Построение ломаной**
```
Начало: (0,0)  
Направление: вправо (0°)

Для i от 0 до len(P_filtered)-2:  
gap = P_filtered[i+1] - P_filtered[i]
 Если P_filtered[i] - младший близнец:
   цвет = зелёный
 Иначе:
   цвет = чёрный
 
 Рисуем отрезок длиной gap в текущем направлении
 Поворачиваем направление на -90°
```


4. **Визуальные маркеры**
- Красная точка: начало (0,0)
- Синяя точка: конец последнего отрезка
- Зелёные линии: отрезки, начинающиеся от младших близнецов

## Математические особенности

### Свойство 1: Ортогональность
Все отрезки строго горизонтальны или вертикальны, что создаёт "квадратную" геометрию.

### Свойство 2: Фильтрационная симметрия
Удаление старших близнецов создаёт неслучайные пропуски в последовательности.

### Свойство 3: Цветовая кодировка
Зелёные линии соответствуют младшим близнецам, позволяя изучать их распределение визуально.

## Теоретическая значимость

Метод преобразует:
- **Числовые свойства** → **Геометрические формы**
- **Статистические закономерности** → **Пространственные структуры**
- **Гипотезы о распределении** → **Визуальные паттерны**

Это открывает новые возможности для интуитивного понимания сложных математических концепций.
