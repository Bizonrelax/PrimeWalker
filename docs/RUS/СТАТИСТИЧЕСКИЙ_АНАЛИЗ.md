# СТАТИСТИЧЕСКИЙ АНАЛИЗ ДЛЯ PRIMEWALKER

## Предлагаемые метрики

### 1. Геометрические метрики
- **Фрактальная размерность** (box-counting метод)
- **Коэффициент заполнения** (отношение занятой площади к описанному квадрату)
- **Центр масс** системы
- **Моменты инерции** относительно осей

### 2. Топологические метрики
- **Количество компонент связности**
- **Длина минимального покрывающего дерева**
- **Коэффициент кластеризации** для вершин
- **Диаметр графа** (максимальное расстояние между вершинами)

### 3. Числовые метрики
- **Распределение длин отрезков** (гистограмма)
- **Автокорреляция** последовательности разностей
- **Среднее и дисперсия** длин отрезков
- **Соотношение зелёных/чёрных отрезков**

### 4. Временные метрики (при анимации)
- **Скорость расширения** рисунка
- **Темп появления новых объектов**
- **Периодичность** в возникновении паттернов

## Методы анализа

### Корреляционный анализ
- Корреляция между длиной отрезка и его цветом
- Корреляция между направлением и последующими отрезками
- Кросс-корреляция с известными последовательностями (простые числа-близнецы и т.д.)

### Кластерный анализ
- Выделение кластеров по цвету
- Кластеризация фигур по форме
- Иерархическая кластеризация объектов

### Ряды Фурье
- Спектральный анализ последовательности разностей
- Выявление периодичностей
- Частотные характеристики системы
