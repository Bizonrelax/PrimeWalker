# ГЛОССАРИЙ ТЕРМИНОВ

## Основные понятия
- **СБ (Старший Близнец)** - большее число в паре простых близнецов (разность = 2)
- **МБ (Младший Близнец)** - меньшее число в паре простых близнецов
- **Линия-портал** - отрезок, который накладывается на другой отрезок, создавая переход между измерениями
- **ССО (Сила Связи Объектов)** - количество линий, которые нужно удалить для разделения объектов

## Геометрические термины
- **Родительский объект** - первичная структура, построенная из начальной последовательности
- **Дочерний объект** - структура, возникающая при наложении линий
- **Измерение** - пространственный слой, в котором существуют объекты
- **Наложение** - ситуация, когда два отрезка проходят по одним координатам

## Статистические параметры
- **Размах в стороны** - максимальное отклонение вправо и влево от начала координат
- **Рост в высоту** - максимальное отклонение вверх и вниз
- **Описанная окружность** - минимальная окружность, содержащая весь рисунок
- **Описанный n-угольник** - минимальный многоугольник с n углами, содержащий рисунок

## Алгоритмические термины
- **Фильтрация СБ** - удаление старших близнецов из последовательности
- **Поворот -90°** - изменение направления по часовой стрелке на 90 градусов
- **Масштабирование** - коэффициент увеличения/уменьшения рисунка
